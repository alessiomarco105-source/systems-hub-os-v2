# Daily Agent Recap

## Purpose

Summarize material agent work, blockers, approvals, and next actions across the company.

## Minimum Context

- run receipts created since the previous recap;
- active project registry;
- new decisions and approvals;
- failed or blocked jobs.

## Output

Separate company, capabilities, and projects. Keep the notification short and place evidence in a dated run receipt. Do not report low-value housekeeping or imply that an unapproved external action occurred.
