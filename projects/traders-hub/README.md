2. `context-map.yaml` for the minimum context required by each task.
3. `reports/current-status.md` for the latest defensible state.

This package contains reviewed project context, not source code, credentials, raw customer data, personal trading records, or company-wide authority. Governance and approvals remain in the Systems Hub OS root.
